Custom URL

Add url property to the category data to set the custom URL. This could be useful to use this plugin as a collapsible page menu

    	{
        name: "Electronics",
        id: 4,
        items: [// Required: array, even empty
          {
            name: "TV",
            id: 40,
            items: [
              ................
              {
                name: "Sony",
                id: 402
              },
              {
                name: "Other - custom URL",
                url: '/app/other'
              }
            ]
          },
        ]
      },
